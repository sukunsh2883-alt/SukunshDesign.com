CodePen setup:

1. Copy index.html into the HTML panel.
2. Copy style.css into the CSS panel.
3. Copy script.js into the JS panel.

Add these external JS resources in this exact order:

https://cdn.jsdelivr.net/npm/gsap@3.12.5/dist/gsap.min.js
https://cdn.jsdelivr.net/npm/gsap@3.12.5/dist/Draggable.min.js

Optional only if you add scroll animations later:

https://cdn.jsdelivr.net/npm/gsap@3.12.5/dist/ScrollTrigger.min.js
